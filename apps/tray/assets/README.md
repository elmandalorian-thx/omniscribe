# Tray Icon Assets

Place the following PNG icons here (16x16 or 32x32):

- `tray-idle.png` — Gray icon, default state
- `tray-recording.png` — Red icon, recording active
- `tray-processing.png` — Yellow icon, transcribing
- `tray-syncing.png` — Blue icon, syncing to cloud
- `tray-error.png` — Orange icon, error state
- `icon.ico` — Windows app icon (256x256)

Generate these with any icon tool or use simple colored circles.
